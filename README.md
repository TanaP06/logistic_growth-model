# Logistic (S-curve) growth Model

This project is a Python-based financial simulation that models **user growth, revenue, costs, and break-even timing** for a SaaS-style product using a **logistic (S-curve) growth model**.

---

## What This Project Does

The model simulates:

- **User Growth**
  - Logistic (sigmoid) growth over time
  - Market saturation behavior

- **Revenue Generation**
  - Freemium monetization model
  - Premium conversion rate
  - Monthly subscription pricing

- **Cost Structure**
  - Infrastructure costs with economies of scale
  - Storage costs per user
  - Development, MVP, and operational costs

- **Profit & Break-Even Analysis**
  - Yearly profit over time
  - Automatic detection of break-even year
  - Visualization of revenue vs. costs

---
